Уроки програмування на Ruby
На основі даних із сайту https://rubyrush.ru/

Автор намагається використовувати наступні правила при написанні коду програм:

- Назви прикладів починаються із великої літери;
- Якщо у назві декілька слів, то наступні слова пишуться з малої літери;
- Якщо у назві декілька слів, то вони розділяються нижнім підкресленням;
- Назви методів починаються із маленької літери;
- Якщо у назві методу декілька слів, то наступні слова пишуться з великої літери;
- Назви змінних пишуться з малої літери;
- Назви змінних, класів та методів у коментарях виділені знаками < та >;
- Назви методів без параметрів пишуться із круглими дужками - напр. test();
- Якщо коментар відноситься до однієї строки коду - перенос на новий рядок не ставиться;
- Якщо коментар відноситься до декількох строк коду - перенос на новий рядок ставиться;
